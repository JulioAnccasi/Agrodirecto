# Agrodirecto
AgroDirecto es una plataforma digital diseñada para optimizar la cadena de suministro agrícola en el Perú, conectando de forma directa a pequeños productores con transportistas mediante un modelo de logística compartida.

Autores (Equipo de Desarrollo):
- Anccasi Bellido, Julio Cesar
- Laurente Benito, Gianira Esther
- López Cruz, Joan Gabriel
- Molocho Mallma, Daniela Alejandra
- Vásquez Rojas, Gadiel Eddie Leonel

Segmentos Objetivo:
- Pequeños Agricultores.
- Transportistas de Carga.

Características Principales:
- Consolidación Inteligente: Agrupamiento de cargas pequeñas mediante algoritmos de geolocalización para reducir costos de flete.
- Perfiles Verificados: Sistema de validación de identidad y reputación para garantizar transacciones seguras.
- Seguimiento en Tiempo Real: Monitoreo GPS de la carga para reducir la incertidumbre del productor.
- Gestión de Cargas de Retorno: Herramientas para que los transportistas eviten viajes vacíos, maximizando su rentabilidad.
